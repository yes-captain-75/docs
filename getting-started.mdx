## Run docs locally

```bash
npx mint dev
```

Open `http://localhost:3000`.

## Check for broken links

```bash
npx mint broken-links
```

## Content rules for this docs site

- Keep docs practical and short.
- Focus on plugin workflows and troubleshooting.
- Skip API docs.
