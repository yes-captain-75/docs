## Getting started

1. To connect Drop Dead to C4D, install our Cinema 4D Bridge from **Settings** -> **Install C4D Bridge…** Restart Cinema 4D.
2. Open Drop Dead.
3. Drag your texture files or folder set into Drop Dead.
4. Drop Dead builds a fully wired StandardRedshift material automatically.
5. If objects are selected, the new material is applied automatically.

## What it does

Drop Dead removes the repetitive node setup work in Redshift. Instead of manually wiring maps in the node editor, you drop textures and get a ready-to-use material.

## Key features

- Drop textures and create a material instantly.
- Auto-apply materials to selected objects.
- Auto-wire texture maps into a complete Redshift shader.
- Skip manual node hookups.

## Requirements

- Cinema 4D R21 or later.
- Redshift 3.0 or later.
- macOS 10.14+ or Windows 10+.

## Troubleshooting

### Material is not applied to my object

- Select your target object before dropping textures.
- If nothing is selected, assign the generated material manually.

### Result does not look right

- Confirm your texture set is complete and belongs to the same material.
- Re-drop a clean texture set with consistent naming.
